Detailed Description

Complete documentation for the Game Engine Python API.

Overview

Comprehensive documentation for all classes and methods available in the Python API. The API provides access to the game engine's core functionality including:

  • Core utilities (vectors, components)
  • Game objects and scene management
  • Input handling and physics
  • Graphics and animation
  • Game logic and scripting

Core Classes

Usage Examples

Basic Game Setup

import engine
# Initialize game
app = engine.GameApplication("My Game")
app.start(False)
# Create player
player = engine.GameObjectFactory.createPlayerTest()
# Add custom behavior
player.addScript(engine.PlayerInputScript())
# Game loop
while app.m_gameIsRunning:
app.input()
app.update()
app.render()
app.advanceFrame()

Scene Creation

enemy_transform = enemy.getComponent(engine.Transform)
